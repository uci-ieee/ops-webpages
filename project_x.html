Overview

There always come a time in life where you choose to challange yourself and this is exactly how you can do so! This project is completely optional, please don't feel that you have to complete this. For those who want a tough challange please attempt this!

This project utilizes a 128x64 LED display to display individual pixels much like your computer monitors. The idea of this project is to utilize the parts we own and research how to complete such a project. We want to be able to make a Etch A Sketch from scratch so when you think about it, what parts do we need? There will be minimal guidance as engineering typically doesn't layout the pieces for you to build, you must also find those pieces too!





Software Installations

Adafruit Libraries

These are the two libraries you need to make the LED display work correctly. Make sure to install the latest version. The correct ones are the ones made by Adafruit and makes sure its LSM6DS and SSD1306. You can find these on the library manager tab and clicking install.











Instructions

What is an Etch A Sketch? Its a kids toy where you draw things using two dials that control a cursor of sorts where one dial changes the horizontal position while the other the vertical position. This should give you a strong hint on what tool should be used here. On top of the basic function of an Etch A Sketch, make a simple button to reset the sketch clearing the board.

A much more difficult project which is also doable with the same LED display is making the google dinosaur game jumping over small obstacles and making a scoring system. You can think of another idea you can use the LED screen for and create that as well! Challenge yourself beyond what's provided on the website and make your own cool ideas!




Below are the library and functions that might be useful 
#include <Wire.h>
#include <Adafruit_GFX.h>
#include <Adafruit_SSD1306.h>

#define SCREEN_WIDTH 128
#define SCREEN_HEIGHT 64
#define OLED_RESET -1

#define I2C_SDA 6  // Custom SDA pin (GPIO6 on ESP32)
#define I2C_SCL 7  // Custom SCL pin (GPIO7 on ESP32)

TwoWire myWire = TwoWire(0);  // Use hardware I2C bus 0

Adafruit_SSD1306 display(SCREEN_WIDTH, SCREEN_HEIGHT, &myWire, OLED_RESET);

void setup() {
	display.clearDisplay();
	display.setTextColor(SSD1306_WHITE);
	Serial.begin(115200);
}
						
void loop() {
	display.drawPixel(xVal, yVal, SSD1306_WHITE); //Draws a pixel at (xVal, yVal)
	display.fillRect(xVal,yVal, width, height, color); //Draw a rectangle starting at the bottom left (xVal, yVal)
  	display.display(); //Displays pixel
}
